One Page Website Template

Written in HTML and CSS

Template Overview

This template was orginially created for the 2022 New Media Caucus SymposiumFuture Bodies, I have adapted my orginal template into a more general one page website template open for anyoine to use under an MIT License. The repository for this template can be found here:

https://github.com/Chelsea-Thompto-Teaching-Examples/one-page-website-temp

Uses

Below, the template mimics a portfolio website but a one page site might be useful for any number of sites such as sites for projects, events, or organizations.

Details

This page relies on HTML tag IDs for navigation. Each section is given an ID that is then linked to the text at the top of the page. You can use hashtags in the "href" section of an "a" tag to link to a given ID. Additionally, the "Back To The Top" button uses CSS to stay on the screen at all times and is linked to an ID at the top of the page.
